# Graph.ist - Personal Website

Personal website and portfolio for a Senior Researcher in Human-Computer Interaction.

## About

This is a minimalist personal website showcasing professional background in HCI, UX, and interface design research. The site features a clean, typography-focused design that emphasizes content and accessibility.

## Features

- **Semantic HTML5** structure for better accessibility and SEO
- **Responsive design** optimized for all devices
- **Open Graph and Twitter Card** meta tags for rich social media sharing
- **Minimalist design** with focus on typography and readability
- **Fast loading** with minimal dependencies

## Technologies

- HTML5
- CSS3 (custom styles)
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) font from Google Fonts

## Structure

```
/
├── index.html      # Main HTML file
├── style.css       # Stylesheet (custom design)
├── CNAME           # Custom domain configuration
└── README.md       # This file
```

## Deployment

This site is designed to be hosted on static hosting platforms like:
- GitHub Pages
- Netlify
- Vercel
- Any static file server

## Local Development

Simply open `index.html` in a web browser, or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

Then visit `http://localhost:8000` in your browser.
